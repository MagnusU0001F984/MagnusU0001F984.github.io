Projects

MagnusChess
C++20 · UCI chess engine · MNUE evaluation
MagnusChess is a UCI-compatible chess engine using classical alpha-beta/PVS search and MNUE evaluation. It supports practical engine testing workflows, Syzygy tablebases, MultiPV, and versioned strength comparisons.
  • Classical search architecture with iterative deepening and PVS.
  • MNUE evaluation with practical UCI engine workflows.
  • Designed for reproducible match testing and engine development.
MagnusMNUE
Neural evaluation resources for MagnusChess
A repository for MNUE evaluation files and related resources used by MagnusChess.

Chess Engine Testing

I use reproducible engine testing workflows with UCI engines, fixed time controls, opening books, and tournament-style match reports. My engine development focuses on practical playing strength, search efficiency, evaluation quality, and implementation robustness.

Topics: alpha-beta search, PVS, move ordering, transposition tables, history heuristics, pruning, MNUE evaluation, Syzygy tablebases, and reproducible testing.
